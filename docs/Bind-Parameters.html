linkBind Parameters

The BindPara class provides utilities of easily creating a bind parameters object to be used in a query. Using the appropriate methods, if a key already exists in the parameter, a different one will be used. So, there is no need of manually ensuring uniqueness of keys.

linkCreating a BindParam instance

First of all, a BindParam instance must be created.

1/* --> creating an empty BindParam instance */

2const bindParam = new BindParam();

3

4/* --> creating a BindParam instance with data (using an object) */

5const bindParam = new BindParam({

6    x: 5,

7    y: 'bar'

8});

9

10/* --> creating a BindParam instance with data (using many objects) */

11const bindParam = new BindParam(

12    {

13        x: 5,

14        y: 'bar'

15    },

16    {

17        z: true

18    }

19);

20

21/* --> the following will throw an error since the keys are not unique. To add non-unique keys, the `getUniqueName` method must be used */

22/* --> this is to prevent unexpected behavior in a case like the following: "n1.x = $x, n2.x = $x", where a different value is intended to be used each time */

23const bindParam = new BindParam(

24    {

25        x: 5

26    },

27    {

28        x: true // -> error! key 'x' is not unique

29    }

30);

linkGetting the values of a BindParam

The get method can be used

1const bindParam = new BindParam({

2    x: 5,

3    y: 'bar'

4});

5

6console.log(bindParam.get()); // { x: 5, y: 'bar' }

linkAdding a value to an existing BindParam instance

The add method can be used, however the added keys must be unique. To add non-unique keys, the getUniqueName method must be used.

1/* --> */

2const bindParam = new BindParam({

3    x: 5

4});

5

6/* --> adding a single object */

7bindParam.add({

8    y: 'bar'

9});

10

11console.log(bindParam.get()); // { x: 5, y: 'bar' }

12

13/* --> adding multiple objects */

14bindParam.add(

15    {

16        z: true,

17        a: 1

18    },

19    {

20        b: 2

21    }

22);

23

24console.log(bindParam.get()); // { x: 5, y: 'bar', z: true, a: 1, b: 2 }

25

26/* --> adding an existing key will result in an error */

27

28bindParam.add({ x: 4 }); // -> error! key 'x' is not unique

linkGetting a unique name to use

The getUniqueName can be used to generate a key which doesn't already exist in the bind param. This key can be used in your queries.

1const bindParam = new BindParam({

2    x: 5,

3    y: 'bar'

4});

5/* --> let's suppose you use those keys in a query. The bindParam can be used as a parameter when we run it */

6let statement = `MATCH (n) WHERE n.x = $x AND n.y = $y`;

7

8/* --> get a unique key based on the 'x' prefix */

9const uniqueName = bindParam.getUniqueName('x');

10console.log(uniqueName); // x__aaaa

11

12/* --> the value for this unique name should be added to the bindParam */

13bindParam.add({

14    [uniqueName]: 4

15});

16console.log(bindParam.get()); // { x: 5, y: 'bar', x__aaaa: 4 }

17

18/* --> this uniqueName can be used in queries */

19statement += `WITH n MATCH (o) WHERE o.x = $${uniqueName} RETURN n, o`;

20

21/* --> we can now run this query */

22await queryRunner.run(statement, bindParam.get());

The getUniqueNameAndAdd can be used to perform the getUniqueName and add methods at once.

1const bindParam = new BindParam({

2    x: 5,

3    y: 'bar'

4});

5/* --> get a unique key based on the 'x' prefix and add the given value to the bindParam */

6const uniqueName = bindParam.getUniqueNameAndAdd('x', 4);

7

8console.log(uniqueName); // x__aaaa

9console.log(bindParam.get()); // { x: 5, y: 'bar', x__aaaa: 4 }

linkCloning a BindParam

A BindParam instance can be cloned, so a new one is returned with the same parameters.

1const oldBindParam = new BindParam({

2    x: 5,

3    y: 'bar'

4});

5oldBindParam.add({ z: true });

6

7console.log(oldBindParam.get()); // { x: 5, y: 'bar', z: true }

8

9const newBindParam = oldBindParam.clone();

10newBindParam.add({ a: 1 });

11oldBindParam.add({ b: 2 });

12

13console.log(newBindParam.get()); // { x: 5, y: 'bar', z: true, a: 1 }

14console.log(oldBindParam.get()); // { x: 5, y: 'bar', z: true, b: 2 }

linkAcquire a BindParam instance

The acquire static can be used to ensure that a BindParam instance is at hand. If one is passed, it will be returned as is. Else, a new one will be created.

1const bindParamFirst = BindParam.acquire(null);

2console.log(bindParamFirst instanceof BindParam); // true

3

4const bindParamSecond = BindParam.acquire(bindParamFirst);

5console.log(bindParamFirst === bindParamSecond); // true

linkRemoving parameters

The remove method can be used to remove keys from the BindParam.

1const bindParam = new BindParam({

2    x: 5,

3    y: 'bar'

4});

5

6const uniqueName = bindParam.getUniqueNameAndAdd('x', 4);

7

8console.log(uniqueName); // x__aaaa

9console.log(bindParam.get()); // { x: 5, y: 'bar', x__aaaa: 4 }

10

11bindParam.remove([uniqueName, 'y']);

12console.log(bindParam.get()); // { x: 5 }
