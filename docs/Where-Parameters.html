linkWhere Parameters

An instance of the Where class can be used to easily create a statement and a bind parameter to be used in a query.

linkCreating a Where instance and getting its values

Creating a Where instance with values for the identifier n

1const where = new Where({

2    /* --> the node identifier is the key */

3    n: {

4        /* --> the per-identifier where values are used */

5        x: 5,

6        y: 'bar'

7    }

8});

9

10/* --> the "where" property has the statement to be used in the query */

11console.log(where.statement); // "n.x = $x AND n.y = $y"

12/* --> the "bindParam" property has a BindParam instance whose values can be used in the query */

13console.log(where.bindParam.get()); // { x: 5, y: 'bar' }

Creating a Where instance with values for the identifiers n and o

1const where = new Where({

2    n: {

3        x: 5,

4        y: 'bar'

5    },

6    o: {

7        z: true,

8    }

9});

10

11console.log(where.statement); // "n.x = $x AND n.y = $y AND o.z = $z"

12console.log(where.bindParam.get()); // { x: 5, y: 'bar', z: true }

Since Where uses a BindParam, non-unique keys can be used which will be automatically associated with a unique key

1const where = new Where({

2    n: {

3        x: 5,

4        y: 'bar'

5    },

6    o: {

7        /* --> since it's for a different identifier, we can use any key we want, even if it's used in another identifier */

8        x: true,

9    }

10});

11

12console.log(where.statement); // "n.x = $x AND n.y = $y AND o.x = $x__aaaa"

13console.log(where.bindParam.get()); // { x: 5, y: 'bar', x_aaaa: true }

An existing BindParam instance can be used, to ensure unique keys. The same instance will be used in the Where instance, so it will be mutated.

1const existingBindParam = new BindParam({

2    x: 4,

3});

4

5const where = new Where(

6    {

7        n: {

8            /* --> the same key as in the bind param can be used */

9            x: 5,

10            y: 'bar'

11        }

12    },

13    existingBindParam

14);

15

16/* --> the "x" key already exists in the bind param, so a new one is used */

17console.log(where.statement); // "n.x = $x__aaaa AND n.y = $y"

18console.log(where.bindParam.get()); // { x: 4, x_aaaa: 5, y: 'bar' }

19console.log(where.bindParam === existingBindParam); // true

An existing Where instance can be used. In this case, the parameters of it will be merged with the one that's being created. The existing Where instance won't be mutated at all.

1const existingWhere = new Where(

2    {

3        n: {

4            x: 4

5        },

6        o: {

7            z: true

8        }

9    }

10);

11

12console.log(existingWhere.statement); // "n.x = $x AND o.z = $z"

13console.log(existingWhere.bindParam.get()); // { x: 4, z: true }

14

15const newWhere = new Where(

16    {

17        n: {

18            y: 'bar'

19        },

20        m: {

21            z: 'foo'

22        }

23    },

24    existingWhere

25);

26

27console.log(newWhere.statement); // "n.x = $x AND n.y = $y AND o.z = $z AND m.z = $z__aaaa"

28console.log(newWhere.bindParam.get()); // { x: 4, y: 'bar', z: true, z__aaaa: 'foo' }

5    }

7

8where.addParams({

linkUsing a Where instance in a query

A Where instance can easily be used in a query using its statement and bindParam properties

1const where = new Where({

2    n: {

3        x: 5,

4        y: 'bar'

5    },

6    o: {

7        z: true,

8    }

9});

10

11await queryRunner.run(

12    `MATCH (n), (o) WHERE ${where.statement} RETURN n, o`,

13    where.bindParam.get()

14);

linkAcquire a Where instance

The acquire static can be used to ensure that a Where instance is at hand. If one is passed, it will be returned as is. If an object with where parameters is passed, a new Where instance will be created with them. Else, a new one will be created.

1const whereFirst = Where.acquire(null);

2console.log(whereFirst instanceof Where); // true

3

4const whereSecond = Where.acquire(whereFirst);

5console.log(whereFirst === whereSecond); // true

6

7/* --> an object with where parameters can be used */

8const whereWithPlainParams = Where.acquire({

9        n: {

10            x: 5

11        }

12});

13console.log(whereWithPlainParams instanceof Where); // true

14

15/* --> a BindParam instance can be passed to be used */

16const existingBindParam = new BindParam({ x: 4 });

17const whereWithBindParams = Where.acquire(

18    {

19        n: {

20            x: 5

21        }

22    },

23    existingBindParam

24);

25

26console.log(whereWithBindParams instanceof Where); // true

27console.log(whereWithBindParams.statement); // "n.x = $x__aaaa"

28console.log(whereWithBindParams.bindParam.get()); // { x: 4, x__aaaa: 5 }
